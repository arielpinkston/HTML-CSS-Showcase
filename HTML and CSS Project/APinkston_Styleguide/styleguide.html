Colors Used

Primary

This color is used for headings and links. rgb(85, 120, 184)

Secondary

This color is mostly used for navigation and. rgb(184, 159, 85)

Accent 1

This color is an accent used for main backgrounds. rgb(255, 252, 214)

Accent 2

This color is an accent used for the backgrounds of different sections. rgb(255, 255, 255). The outline color surounding it is used for outlining subsections. rgb(182, 182, 182)

Accent 3

This color is an accent used for for all regular text and for highlighted links. rgb(111, 104, 0)
